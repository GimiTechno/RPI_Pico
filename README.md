# 【RPI_Pico(公開:開発用)】
 
個人でのRaspberry Pi Picoの開発Gitリポジトリ
 
## 【機能(ドライバ等)】
 
・マルチコア ... 5bit DAC
 
・CLK ... 125MHz駆動。ペリフェラルクロック表示機能あり
 
・WDT ... 8sec OVF
 
・Timer ... ワンショットタイマ
 
・PIO ... 5bit DAC
 
・UART ... 自前モニタプログラム(FSYMonitor)
 
・SPI ... BME280ドライバ
 
・I2C ... DS3231 RTC、PCF8574A(LCD1602 I2C化 IC）
 
・ADC ... DMAでのワンショットスキャン


### 【言語】
 
Cのみ。C++も予定。
